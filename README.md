# HR Analytics Dashboard - Employee Attrition Analysis 📊

## 📌 Project Objective
The goal of this project is to analyze employee attrition data to identify key factors that lead to employee turnover. By visualizing various HR metrics, this dashboard provides actionable insights to help the organization improve retention strategies.

## 🛠️ Tools Used
- **Power BI Desktop:** For data visualization and dashboard creation.
- **Power Query:** For data cleaning and transformation.
- **DAX:** Used for creating calculated measures and KPIs.

## 💡 Major Insights
- **Salary Impact:** Employees in the **"Upto 5k"** salary slab show the highest attrition, indicating that lower-income brackets are more prone to leaving.
- **Job Role Vulnerability:** **Laboratory Technicians** and **Sales Executives** have the highest attrition counts (60 and 55 respectively).
- **Age Group:** The **26-35** age group is the most volatile, representing the highest number of departures.
- **Education Background:** Over **63%** of the attrition comes from employees with **Life Sciences** and **Medical** degrees.
